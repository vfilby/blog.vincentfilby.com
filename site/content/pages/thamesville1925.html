There is a lot of data here:

  • 89 images from a 15MP dSLR
  • each map with all the stitching data and masking is around 1.5GB
  • over 800MB of 256x256 tiles
  • A compressed JPEG of the map totals 50MB
So I don't offer direct downloads for the source images. If you want the images contact me and we can arrange something.
